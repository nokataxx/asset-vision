**作成日**: 2026年1月29日
**更新日**: 2026年2月1日（MSCI ACWIベースに変更）

---

## 1. 概要

本ドキュメントは、金融資産シミュレーターのレジーム設定デフォルト値が、過去の市場実績データと整合しているかを検証した結果をまとめたものである。

### 検証の観点

1. **MSCI ACWI（オルカン）** を主要な参照データとして採用
2. 暴落+戻り期シナリオモデルとしての妥当性
3. 全世界株式インデックス投信を前提とした設定としての適切性

### ベンチマーク選定理由

- 日本の個人投資家に人気の「オルカン」（eMAXIS Slim 全世界株式）はMSCI ACWIに連動
- S&P 500より分散効果があり、やや低リターン・低リスク
- 長期積立投資のベンチマークとして適切

### 4.2 長期期待リターンの検証

現在のパラメータ設定での長期期待リターンを定常分布から計算：

```
回復年数の推定:
  暴落後残高比率 = 1 + (-0.24) = 0.76
  回復年数 = log(1/0.76) / log(1.22) ≈ 1.4年

定常確率の計算:
  p = 0.10（暴落確率）
  n = 1.4（回復年数）
  分母 = 1 + p × (1 + n) = 1 + 0.10 × 2.4 = 1.24

  π_normal = 1 / 1.24 = 80.6%
  π_crash = 0.10 / 1.24 = 8.1%
  π_recovery = (1.4 × 0.10) / 1.24 = 11.3%

期待リターン:
  = 0.806 × 12% + 0.081 × (-24%) + 0.113 × 22%
  = 9.67% - 1.94% + 2.49%
  ≈ 10.2%
```

**注**: この期待リターン11.1%は算術平均であり、実際のCAGR（複利成長率）はボラティリティドラッグにより低くなる。MSCI ACWIの実際の分布（通常期57%、暴落期17%、戻り期26%）で計算すると約8.9%となり、全体平均と整合する。

### 4.4 暴落+戻り期モデルの評価

本シミュレーターのレジームモデル特性：

| 特性 | 内容 |
|------|------|
| 暴落期の定義 | 年次リターンが-10%以上の下落年に相当 |
| 暴落期の継続 | 1年で終了、翌年から自動的に戻り期へ遷移 |
| 戻り期中の再暴落 | 10%の確率で発生可能 |
| 戻り期の終了条件 | 株式残高が暴落前水準に回復した時点 |

**複数年にわたる暴落局面の表現**:
- 実際のベアマーケットは平均10ヶ月継続
- 本モデルでは「暴落→戻り→再暴落」の連鎖で複数年の下落局面を表現可能
- 戻り期中も10%の確率で再暴落が発生するため、長期低迷シナリオも再現される

## 5. 結論

### 5.1 総合評価

**現在のデフォルト値はMSCI ACWI（オルカン）の歴史的データと整合しており、適切である。**

全世界株式インデックス投信の長期運用シミュレーションには妥当な設定である。

### 5.2 設定の特徴

| 特徴 | 内容 |
|------|------|
| ベンチマーク | MSCI ACWI（オルカン） |
| 全体平均リターン | 約8.9%（S&P 500より約1%低い） |
| リスク水準 | S&P 500より低ボラティリティ |
| 保守性 | 標準偏差を実測値よりやや高めに設定 |

### 5.3 推奨事項

1. **現在のデフォルト値を維持**する
2. ユーザーには**25%タイルを基準**に計画を立てることを推奨
3. S&P 500をベンチマークとする場合は、通常期利回りを12-14%に引き上げることを検討
