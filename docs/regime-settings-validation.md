## 1. 概要

本ドキュメントは、金融資産シミュレーターのレジーム設定デフォルト値が、過去の市場実績データと整合しているかを検証した結果をまとめたものである。

### 検証の観点

1. S&P 500およびMSCI ACWIの歴史的データとの整合性
2. 暴落+戻り期シナリオモデルとしての妥当性
3. インデックス投信を前提とした設定としての適切性

### 4.2 長期期待リターンの検証

現在のパラメータ設定での長期期待リターンを定常分布から計算：

```
回復年数の推定:
  暴落後残高比率 = 1 + (-0.22) = 0.78
  回復年数 = log(1/0.78) / log(1.18) ≈ 1.5年

定常確率の計算:
  p = 0.13（暴落確率）
  n = 1.5（回復年数）
  分母 = 1 + p × (1 + n) = 1 + 0.13 × 2.5 = 1.325

  π_normal = 1 / 1.325 = 75.5%
  π_crash = 0.13 / 1.325 = 9.8%
  π_recovery = (1.5 × 0.13) / 1.325 = 14.7%

期待リターン:
  = 0.755 × 10% + 0.098 × (-22%) + 0.147 × 18%
  = 7.55% - 2.16% + 2.65%
  ≈ 8.0%
```

**評価**: S&P 500の長期CAGR（年複利成長率）6-8%と整合しており、適切である。

### 4.3 暴落+戻り期モデルの評価

本シミュレーターのレジームモデル特性：

| 特性 | 内容 |
|------|------|
| 暴落期の定義 | 年次リターンが-10%以上の下落年に相当 |
| 暴落期の継続 | 1年で終了、翌年から自動的に戻り期へ遷移 |
| 戻り期中の再暴落 | 13%の確率で発生可能 |
| 戻り期の終了条件 | 株式残高が暴落前水準に回復した時点 |

**複数年にわたる暴落局面の表現**:
- 実際のベアマーケットは平均10ヶ月継続
- 本モデルでは「暴落→戻り→再暴落」の連鎖で複数年の下落局面を表現可能
- 戻り期中も13%の確率で再暴落が発生するため、長期低迷シナリオも再現される

## 5. 結論

### 5.1 総合評価

**現在のデフォルト値は全体的に適切であり、歴史的データと整合している。**

requirements.mdに記載された根拠（S&P 500およびMSCI ACWIの過去実績）とも一致しており、インデックス投信の長期運用シミュレーションには妥当な設定である。

### 5.2 改善候補

唯一の改善候補として、**通常期標準偏差を10%から12-14%に引き上げる**ことが挙げられる。

| 現在値 | 改善案 | 影響 |
|--------|--------|------|
| 10% | 12-14% | 通常期のマイナスリターン年の確率が上昇し、悲観シナリオがより保守的になる |

ただし、以下の理由から現状維持でも問題ない：
1. 暴落期を別カテゴリとして分離するモデル設計
2. 25%タイルを基準に保守的な計画を立てる運用推奨
3. ユーザーが手動で調整可能

### 5.3 推奨事項

1. **現在のデフォルト値を維持**する
2. ユーザーには**25%タイルを基準**に計画を立てることを推奨（既にrequirements.mdに記載済み）
3. 将来的に標準偏差の調整を検討する場合は、通常期を12%程度に引き上げることを検討
